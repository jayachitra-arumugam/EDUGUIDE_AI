# EDUGUIDE_AI 📚

EDUGUIDE_AI is an AI-powered educational guidance system that helps students get smart learning recommendations and academic support.

---

## 🚀 Features

* AI-based learning guidance
* Course recommendations
* Student-friendly interface
* Smart educational support
* Responsive web application

---

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Machine Learning

## ⚙️ Installation

### Clone the repository

```bash id="fcr5nr"
git clone https://github.com/jayachitra-arumugam/EDUGUIDE_AI.git
```

### Go to project folder

```bash id="z1trdb"
cd EDUGUIDE_AI
```

### Install dependencies

```bash id="9dr49w"
pip install -r requirements.txt
```

### Run the project

